In recent years, there has been a very rapid evolution of all the components of gaming consoles, especially in the hardware. Both Sony and Microsoft have made drastic changes that have brought incredible success to the market.

PS2

Game console Sony PlayStation 2 (PS2). Supports DVD-ROM and CD-ROM. Toshiba Processor and Sony Emotion Engine 300 MHz. It has built-in RDRAM memory (RAMBUS) of 32 MB capacity, 40 MB cache memory. Flash memory card of 1.8 MB capacity. Integrated 4 MB video card memory, 2D / 3D graphics acceleration, fog effects and 90 ° lighting.

PS3

The PlayStation3 was presented with a CPU called: Cell Broadband Engine (1 x PPE @ 3.2 GHz e 8 x SPE) and a graphics processing unit (GPU) RSX 'Reality Synthesizer' codeveloped by Nvidia and Sony. It was settled with a main memory of 256MB XDR main RAM and an embedded VRAM (Video RAM) 256MB GDDR3 VRAM. The storage occurred in a Hard Drive Disk (HDD) of 80 GB capacity. Few years later Sony launched 2 more models the PS3 SLIM and the PS3 SUPER SLIM, the only main update was the design and the organisation of components in the case.

PS4

The PlayStation4 was presented with a CPU called AMD Jaguar, clock-cycle of 1.64GHz with 8 cores. The graphics processing unit (GPU) changed the world of graphics in gaming, a 1.84 TFLOP AMD Radeon (18CU, 800MHz). It was developed with an 8GB GDDR5 RAM. The size Hard Drive Disk (HDD) was 500GB and the Disk speed of 5400rpm, which had a really good effect on loading and storing data. Sony again released 2 more models the PS4 SLIM and the PS4 PRO. The PS4 SLIM was designed slimmer but with the same hardware components as PS4. The console which changed the hardware and the performance completely is the latest model PS4 PRO with the following specifications.

  • CPU: 2.1GHz 8-core AMD Jaguar
  • GPU: 4.2 TFLOP AMD Radeon (36CU, 911MHz)
  • MEMORY: 8GB GDDR5 + 1GB
  • STORAGE: 1TB

XBOX

The gaming console XBOX was released with a CPU Intel Pentium III – 733 MHz and a Graphics Processing Unit (GPU) NVDIA NV2A – 233 MHz . A 64 MB DDR SDRAM (synchronized dynamic random-access memory). The first Xbox’s storage was limited to 8 GB HDD.

The Xbox 360 was out in 3 different models: XBOX 360, XBOX 360 S, XBOX 360 E. RAM and the Hard Disk Drive were same in all the models but the CPU and GPU were different and gradually improved in each model.

XBOX 360

CPU: Microsoft XCPU (Xenon), 3 cores – 500 MHz
GPU: R500, Xenos chip by ATI – 500 MHz
MAIN MEMORY: 512 MB GDDR3 RAM
HARD DRIVE: 250 GB HDD

XBOX 360 S

CPU/GPU: Custom ATI-designed, combo single chip
MAIN MEMORY: 512 MB of GDDR3 RAM
HARD DRIVE: 250 GB HDD

XBOX 360 E

CPU: IBM custom designed Xenon – 3 cores
GPU: ATI Xenos – 10 MB eDRAM
MAIN MEMORY: 512 MB GDDR3 RAM
HARD DRIVE: 250 GB HDD 2.5′ SATA

The Xbox One is available in 3 different models: XBOX ONE, XBOX ONE S, XBOX ONE X. Microsoft has made minor changes in each model up to the latest model Xbox One X which competes the PS4 PRO.

XBOX ONE

CPU: AMD 8 Core APU – 1.75 GHz
GPU: AMD RadeonGCN architecture, 853 MHz
MAIN MEMORY: 8 GB DDR3
HARD DRIVE: 500 GB HDD

XBOX ONE S

CPU: AMD 8-core APU (2 quad-core Jaguar modules)
GPU: AMD Radeon 914 MHz
MAIN MEMORY: 8 GB DDR3
HARD DRIVE: 1TB HDD

XBOX ONE X

CPU: AMD 8-core APU – 2.3 GHz
GPU: AMD RadeonGCN architecture 1.172 GHz
MAIN MEMORY: 12 GB GDDR5
HARD DRIVE: 2TB HDD
