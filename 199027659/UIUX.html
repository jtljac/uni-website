PS2 User Interface

PlayStation 2 kick started their franchise with a basic user interface which only had two options for the user to choose from. As the system starts up, it shows the user the name of the console in the middle of the screen (Sony Computer Entertainment) then proceeds to the home screen of the gaming console. The colours used within the UI is very consistent throughout the system, as they’ve gone for a dark theme using dark colour such as: Black, Dark Blue and white for the writing to standout.

PS3 User Interface

Sony released the third generation of PlayStation in 2006. The PS3 UI was a very simple column and rows format, which had all the information needed within a UI. From left to right the 10 tabs consist of; User (Profiles), Settings, Pictures, Music, Videos, TV and Video Services, Games, Internet, PS Store and Friends.

PS4 User Interface

PlayStation 4 was released in 2013, alongside their rivals Microsoft. Sony also change the design of the UI. Firstly, Sony decided to only have two rows for the UI, the top row is where the user was able to access important features like; Friends, Settings, Messages, Party Chat, PS Plus, Notification etc. The second row contained all the application which the user could access including Games, PS Store, Internet, Spotify Pictures, Videos (Inc VR), Library.

Xbox Original User Interface

Microsoft started off their Xbox franchise with a basic user interface, which only contained only four options “Memory, Music, Xbox Live and Settings. As the user moves from one option to another the UI create a Science fiction animation and sound to enhance the user experience and make them feel like their operating on a futuristic system.

Xbox 360 User Interface

The Xbox 360 console was released in 2005. Microsoft started their product with a starting screen to display a new generation of gaming. The Xbox 360 dashboard went through a huge amount of change throughout the years of their initially release date of the Xbox 360. The first design of the dashboard was announced as “Blades Dashboard”.

Xbox One User Interface

In 2013 Microsoft released their latest generation called the “Xbox One”. As Microsoft kept the same layout as they had in the previous generation, there were still new features and couple exclusive content which was implanted into the user interface. With the Xbox One first design of the UI there were only three tabs available to the users.
