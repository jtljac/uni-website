In recent years, there has been a very rapid evolution of all the components of gaming consoles. Both sony and microsoft have made drastic changes that have brought incredible success to the market.

PS2

Game console Sony PlayStation 2 (PS2). Supports DVD-ROM and CD-ROM. Toshiba Processor and Sony Emotion Engine 300 MHz. It has built-in RDRAM memory (RAMBUS) of 32 MB capacity, 40 MB cache memory. Flash memory card of 1.8 MB capacity. Integrated 4 MB video card memory, 2D / 3D graphics acceleration, fog effects and 90 ° lighting.

PS3

CPU: Cell Broadband Engine (1 x PPE @ 3.2 GHz e 8 x SPE)
GPU: RSX
MAIN MEMORY: 256MB XDR Main RAM
EMBEDDED VRAM: 256MB GDDR3 VRAM
HARD DRIVE DISK: 2.5” Serial ATA (80 GB HDD)

PS4

MEMORY: 8GB GDDR5
STORAGE: 500GB (5400rpm)

PS4 SLIM

CPU: 1.6GHz 8-core AMD Jaguar

PS4 PRO
